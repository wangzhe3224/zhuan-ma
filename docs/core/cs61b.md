# CS61B: Data Structures and Algorithms

这是一个门中规中矩的数据结构和算法的入门课。实现语言是 Java，但是对于没有 Java 经验的学生来说，
只要进行其他语言的编程，这门课提供的 Java 入门教学足够完成该课程的所有 lab 和作业了。

关于 Java 的部分包括：

- 类和对象
- 递归
- 接口和继承
- 多态
- 高阶函数
- 迭代器、异常等等

数据结构和算法方面包括：

- 抽象数据类型
- 链表、列表
- 树，B-Tree，红黑树
- 散列
- 堆和队列
- 图和图算法
- 排序
- 搜索
- 压缩算法
- 软件工程
- NP 问题

[课程链接](https://sp21.datastructur.es/)
