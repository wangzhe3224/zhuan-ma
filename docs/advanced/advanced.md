# 进阶资源

!!! note
    在完成 [核心课程](../core/core.md) 之前不需要考虑这里的内容。

这里的”进阶“仅仅是针对转码的同学而言，并不是说这里的内容属于计算机科学的进阶内容，相反这里提到的
资源和课程更多属于计算机本科的基础课程。同样，这里我们仍然秉着精简的方针，每一个项目只有一门课或
资源。其实，网上众多资源无论哪一个能够跟下来，都可以掌握该主题的主要框架和内容。
